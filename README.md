# Evolution Chain

Русская браузерная idle-игра про эволюцию сложности материи: от атомов к молекулам, органике, протоклеткам и первой жизни.

## Статус

MVP 0.1: статический сайт для GitHub Pages.

## В игре уже есть

- ресурсы: материя, энергия, сложность, стабильность, информация, биомасса;
- 8 стадий эволюции;
- дерево открытий;
- ручные действия игрока;
- пассивный прирост ресурсов;
- случайные события среды;
- сохранение прогресса в браузере;
- сброс прогресса.

## Стадии

1. Атомы
2. Простые молекулы
3. Органические молекулы
4. Полимеры
5. Самокопирование
6. Протоклетки
7. Первые клетки
8. Одноклеточная жизнь

## Локальный запуск

Открой файл `index.html` в браузере.

## GitHub Pages

В настройках репозитория включи Pages из ветки `main`, папка `/root`.
